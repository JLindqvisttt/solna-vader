# 🌤️ Solna Väder

En enkel hemsida som visar aktuellt väder för **Solna, Stockholm** i realtid.

## Features

- Temperatur + känns-som
- Väderikon & beskrivning
- Luftfuktighet, vindstyrka, molntäcke
- Uppdateras automatiskt var 10:e minut
- Ingen API-nyckel behövs (använder [Open-Meteo](https://open-meteo.com/))

## Öppna lokalt

Öppna bara `index.html` i en webbläsare – inget byggesteg behövs.

## Deploy

Fungerar direkt med **GitHub Pages**:
1. Gå till Settings → Pages
2. Sätt source till `main` / `root`
3. Klart!
